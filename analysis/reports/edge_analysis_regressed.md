# Model Combination Analysis: Spread & Total Edges

Analysis of model combination accuracy at different edge thresholds.

## Methodology

For spreads and totals, edges are calculated using probability-space regression:

1. Compute raw model prediction (average of selected models)
2. Look up implied probability from historical data
3. Regress toward market: `p_final = 0.4 × p_model + 0.6 × 0.50`
4. Edge = p_final − 52.38% (break-even at -110)

For moneylines:
1. Compute median win probability across selected models
2. Regress toward market: `p_final = 0.5 × median(models) + 0.5 × market_implied`
3. Edge = p_final − market_implied
4. Only bet when edge > 0 (positive edge only - we don't bet the other side)
5. ROI uses actual posted opening moneyline odds

**Model abbreviations**: KP=Kenpom, BT=Barttorvik, EM=EvanMiya, HA=Hasla
