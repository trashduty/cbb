# Totals Analysis - UNREGRESSED

## Methodology

### Step 1: Calculate Combo Total
- For each model combination, compute the **mean** of the selected model totals
- Example: KP+BT+EM combo_total = (kenpom_total + barttorvik_total + evanmiya_total) / 3
- Round to nearest 0.5 for lookup

### Step 2: Look Up Probability
- Use `totals_lookup_combined.csv` with inputs:
  - `spread_category`: 1 (spread 0-2.5), 2 (spread 2.5-10), 3 (spread 10+)
  - `market_total`: Opening total line (rounded to 0.5)
  - `model_total`: Combo total from Step 1
- Returns `over_prob` or `under_prob`

### Step 3: Calculate Edge (UNREGRESSED)
- **Edge = lookup_prob - 52.38%**
- 52.38% is break-even at -110 odds
- NO regression toward 50% is applied (this is the unregressed version)

### Step 4: Bet Direction
- If edge > 0: Bet OVER (or UNDER depending on which we're analyzing)
- If edge < 0: Bet the opposite direction
- Filter: Only bet when |edge| >= threshold

### Step 5: Calculate ROI
- Standard -110 juice: Win pays 100/110, loss costs 1
- ROI = ((wins × 0.909) - losses) / total_bets × 100

### Filters Applied
- Home team rows only (one bet per game)
- Games must have: opening_total, opening_spread, over_hit/under_hit
- All models in the combination must have data for that game
