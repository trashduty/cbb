# Moneyline Edge Analysis (REGRESSED)

## Methodology

- Edge = (0.5 × median(model_probs) + 0.5 × market_implied) - market_implied (regressed toward market)
- Positive edge only
- ROI uses actual posted opening moneyline odds

**Note:** Barttorvik only has 55% data coverage, limiting BT combo sample sizes.
