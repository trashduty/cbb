# Edge-Based Model Combination Analysis (Probability-Space Regression)

Analysis of model combination accuracy at different edge thresholds.

## Methodology

**NEW APPROACH**: Regress in probability space AFTER lookup, not before:

- **Spreads/Totals**: RAW model line → lookup probability → regress prob (40% model + 60% × 0.50) → edge
- **Moneylines**: model_prob - market_implied_prob = edge (unchanged)

Example: If lookup returns 60% cover probability:
```
p_final = (0.60 × 0.4) + (0.50 × 0.6) = 0.24 + 0.30 = 0.54
edge = 0.54 - 0.5238 = +1.62%
```

- **ROI**: Standard -110 juice for spreads/totals, actual odds for moneylines
- **Models**: KP=Kenpom, BT=Barttorvik, EM=EvanMiya, HA=Hasla
